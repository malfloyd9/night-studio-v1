# Night Studio v1
A local-only fullscreen creative workspace for focused work and idea capture.
Night Studio v1 is not a productivity app. It's a controlled, immersive environment — a 2AM desk by a rain-streaked window — designed to create psychological separation from the noise of the day and support consistent creative work.

What It Is

A calm, fullscreen 2D scene with ambient rain and desk objects
A lightweight space for capturing ideas and tracking a short task list
A ritual-based environment with intentional open and close states
Local-only, no cloud, no accounts, no tracking

What It Is Not

A task management system
A SaaS product
A productivity dashboard


Stack

React + Vite
Tailwind CSS
Zustand (state management)
Howler.js (audio)
localStorage (persistence)

Design Principles

Calm > clever
Restraint > complexity
One overlay active at a time
No red alerts, no flashing elements, no gamification
Atmospheric objects remain visual-only in V1


Roadmap
V1 is intentionally minimal. No feature additions before 30 days of real use.
Future versions (V2+) may include dictation, AI session tracking, guided breathing, and context-aware quotes — but only after V1 is proven in practice.
